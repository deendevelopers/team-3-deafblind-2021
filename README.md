# team-3-deafblind-2021

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the Expo tool

```bash
  yarn start
```

Follow the prompts in the command line to emulate on either iOS or Android
